
            С локального репозитария на github(первый раз)

  1.Передача файлов с лакального репозитария Ctest на компе
    в удалённый на github , предворительно нужно создать на
    github удалённый репозитарий с таким же именем 
    как и на компе т.е.Ctest


  2.На компе из git-bash зайти ~/Ctest и набрать:

     git init                          //инициализация git в данной директории

     git add "название одного файла"   //добавление файла для commit,
                                         если их много и надо
                                         все сразу (git add . )

     git commit -m "название комита"   //делаем commit в локальный репозитарий

     git branch -M main // переименовывает ветку master на локальном репозитории
                           в main. Таким образом, изменения, произошедшие на 
                           удаленном репозитории больше не конфликтуют 
                           с локальным хранилищем, в котором главная ветка
                           стала также называться main.

     git remote add origin git@github.com:lio68/Ctest.git //команда, которой устанавливается 
                                                            подключение к удаленному серверу 
                                                            github и git репозиторию на нем.
                                                            В данном случае по ssh    

     git push -u origin main //  позволяет запушить (отправить)
                                 ветку main на сервер origin.




             Если уже существует и локальный и удалённый репозитарий: 


      git remote add origin git@github.com:lio68/Ctest.git
      git push
